# Missile Command - 1980 Archive

A browser-based simulation of the classic **Missile Command** arcade game originally developed by Atari in 1980. This repository contains the HTML, CSS, and JavaScript files rendering the classic CRT aesthetic and the core mechanics of defending cities against an endless nuclear missile barrage.

## Features

- **Classic 1980s Arcade Aesthetic**: CRT scanline effects and a retro pixel-art style powered by the Press Start 2P font.
- **Simulation Gameplay (Demo)**: A simplified browser-based emulation of the game mechanics where you can launch interceptors via trackball or mouse clicks to protect your 6 cities from enemy ICBMs.
- **Mission Briefing (Tutorial)**: Instructions detailing game mechanics, interceptors, explosions, MIRVs, and game-over conditions.
- **Hall of Fame**: An archive view of the top gamers and high scores from the early 1980s.

## How to Play

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Navigate to the **LIVE DEMO** tab to start playing the interactive simulation.
4. Click anywhere on the radar screen to launch interceptors to that location.
5. Create explosion clouds to destroy incoming enemy missiles before they obliterate your cities!

## Files Structure

- `index.html` - The main UI framing the simulation, navigation tabs, and content sections.
- `style.css` - Stylesheet defining the CRT look, neon colors, page layout, and game-specific UI.
- `script.js` - Game engine script responsible for the logic rendering the interactive demo on the `<canvas>`.
